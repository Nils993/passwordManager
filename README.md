Менеджер паролей
Реализованая функциональность:

- Списка сохранённых записей
- Добавление новой учётной записи
- Редактирование существующей записи
- Удаление записи
- Скрытие/отображение пароля
- Копирование пароля в буфер обмена
- Поиск
- Группировка по меткам

Стек:

- Vue
- TypeScript
- Pinia
- Composition API
- Vite
- localStorage
- Tailwind

Для запуска проекта в режими разработчика введите команду в консоли:

- npm run dev
  Для предварительного просмотра проекта введите команду в консоли:
- npm run preview
